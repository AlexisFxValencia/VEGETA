What is VEGETA ?

VEGETA is a very light app to plot 3D-Geometries of neutronic codes and export them as 3D files. It doesn't require an installation, it works as a local website. For now, only openMC and MORET are available. SERPENT2 will be added at some point.

Basics :

Principle : Copy-paste your text input on the left, click on "Plot" and wait for the generation of your 3D view

Auto-detect of neutronic code is selected by default but you can manually define the neutronic code used.

The name of the mesh cell and its material is displayed on top of the viewer when the mouse hovers on it

Make a cell transparent :

    You can make a mesh transparent by :
  • right clicking on it
  • click on the checkbox on the left (scroll down) corresponding to the name of the volume

Generate a 3D export :

nom only .ply available (includes colors)
If you want multiple 3D pieces, you have to adapt the view to the export you want, that is make every unecessary cells transparent.
